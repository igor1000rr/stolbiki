# Backgrounds

Фоновые сцены для игры — 5 уникальных картин.

## WebP варианты (для bg_city_day — базовый фон)

| Файл | Размер | Когда служит |
|------|---------|----------------|
| mobile-portrait.webp  | 1080×1672 / 174KB | телефон вертикально |
| mobile-landscape.webp | 1920×1080 / 133KB | телефон горизонтально |
| tablet-portrait.webp  | 1536×2048 / 144KB | iPad вертикально |
| tablet-landscape.webp | 2048×1536 / 122KB | iPad горизонтально |
| desktop.webp          | 2560×1440 / 142KB | ноуты, мониторы до 2K |
| desktop-4k.webp       | 3840×2160 / 226KB | retina, 4K |

## SVG сцены (для остальных скинов)

Векторные, легкие (5-16 КБ), чёткие на любом разрешении:
- **city-night.svg** (16KB) — ночной город, луна, звёзды, светящиеся окна, отражение на воде
- **mountains.svg** (6KB) — три горных гряды, снег, ельник, утреннее небо, солнце в тумане
- **desert.svg** (5KB) — дюны, закатное солнце, кактусы сагуаро, птицы, рябь на песке
- **space.svg** (8KB) — туманности, звёзды, большая планета с кратерами, Сатурн с кольцами

## Совокупные размеры

- 6 WebP (только bg_city_day) — 940 КБ суммарно, но на сессию грузится только 1 (под размер экрана)
- 4 SVG (остальные фоны) — в сумме 35 КБ. Меньше одной картинки.

## Регенерация

WebP из PNG:
```
cwebp -q 82 -m 6 input.png -o output.webp
```

SVG редактируются как обычный XML. Превью: `rsvg-convert -w 1920 file.svg -o file.png`
