# Backgrounds

Фоновые сцены для игры — 2 темы: **День** и **Ночь**.

## WebP варианты для каждой темы

Каждая тема имеет 6 размеров — браузер грузит только один под разрешение/ориентацию.

| Файл | Размер | Когда служит |
|------|---------|----------------|
| `{theme}-mobile-portrait.webp`  | 1080×1920 | телефон вертикально |
| `{theme}-mobile-landscape.webp` | 1920×1080 | телефон горизонтально |
| `{theme}-tablet-portrait.webp`  | 1536×2048 | iPad вертикально |
| `{theme}-tablet-landscape.webp` | 2048×1536 | iPad горизонтально |
| `{theme}-desktop.webp`          | 2560×1440 | ноуты, мониторы до 2K |
| `{theme}-desktop-4k.webp`       | 3840×2160 | retina, 4K |

Где `{theme}` — `day` или `night`.

## Совокупные размеры

12 WebP суммарно ~1.8 MB, но на сессию грузится только 1 (под размер экрана) → ~150 KB средняя нагрузка.

## Регенерация

```
cwebp -q 82 -m 6 input.png -o output.webp
```
